# High-Fidelity Rotorcraft Flight Dynamics Simulation

## Overview

This project presents a nonlinear helicopter flight dynamics simulation framework validated using real flight-test data.

## Features

- Nonlinear rotorcraft dynamics
- Trim analysis
- Flight trajectory simulation
- Stability analysis
- Controller integration

## Validation

Simulation results were validated using real helicopter flight-test data.

## Research Impact

This project served as the foundation for subsequent research on:

- Autonomous VTOL landing
- Vision-based control
- AI-assisted flight systems
- Novel eVTOL/PAV design
